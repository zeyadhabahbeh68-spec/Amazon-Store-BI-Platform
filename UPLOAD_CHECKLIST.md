# Upload Checklist

Upload these files/folders into the GitHub repository:

## Root
- README.md
- requirements.txt
- app1.py
- GitHub_Upload_Guide.md
- UPLOAD_CHECKLIST.md

## Folders
- docs/
- images/
- data/
- dashboards/
- notebooks/
- models/

## Important
Markdown files inside `docs/` use image links like:

```md
![Figure Title](../images/image_name.png)
```

So the `images/` folder must be uploaded at the repository root.
