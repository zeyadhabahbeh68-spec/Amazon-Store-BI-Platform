# GitHub Upload Guide

Use this structure when uploading the project to GitHub:

```text
Amazon-Store-BI-Platform/
├── README.md
├── src/
│   └── app1.py
├── data/
│   ├── amazon(1).csv
│   ├── amazon(2).csv
│   ├── calculation_summary.csv
│   └── product_level_calculations.csv
├── images/
│   └── all figures and charts
└── docs/
    ├── 00_Abstract_and_Keywords.md
    ├── 01_Introduction.md
    ├── 02_Project_Management.md
    ├── 03_Literature_Review.md
    ├── 04_Research_Methodology.md
    ├── 05_Results_and_Discussion.md
    ├── 06_Conclusion.md
    ├── 07_Appendix.md
    └── 08_References.md
```

Upload the files into the matching folders. Markdown documentation should be uploaded instead of Word files.
